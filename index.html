Page d'accueil du miniprojet

Description

Ce miniprojet nous a permis d'acquérir une première expérience concrète en préparation du projet principal. Le travail s'est déroulé de manière itérative, avec des fonctionnalités ajoutées au fil des séances. Pour consulter les résultats finaux sous forme de tableau, cliquez ici

Démarche et Évolution

Le développement s'est articulé autour de plusieurs phases successives :

  1. Le script analysait des URLs pour extraire leur code HTTP, l'encodage utilisé et effectuait un décompte du nombre de mots
  2. Nous avons ensuite organisé ces données dans un tableau pour une meilleure lisibilité
  3. Génération automatique d'un fichier HTML à partir des résultats obtenus
  4. Intégration du framework CSS Bulma pour une présentation claire et responsive
